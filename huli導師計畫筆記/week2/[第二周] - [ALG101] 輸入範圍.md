# [第二周] - [ALG101] 輸入範圍
### 空間估算
int: 4 bytes
double: 8 bytes
JS 中的 Number：8 bytes

### 一百萬個數字
(8 * 1e6) / 1024 = 7812 KB
= 7.6 MB

### 十億個數字
(8 * 1e8) / 1024 = 781200 KB
= 7600 MB = 7.4 GB

### 型態限制
int： -2147483648 ~ 2147483647
JS 數字： Number.MAX_SAFE_INTEGER
浮點數精準度問題
