# gulp 與 webpack 的異同
![](https://i.imgur.com/SP1jADt.jpg)

### gulp -> tack manager
基本上 gulp 可以用 plugin 的方式做任何事，如爬蟲、作業自動化等，但他無法達到 webpack 的 bundle 功能

### webpack -> bundler
webpack 基本上就是把各個 plugin 跟 loader 一起 bundle 成一個檔案，所以在前端開發基本上就是完成案子後把 src bundle 成一個最終檔案交出去
