### emmet
針對 html 程式編輯器的外掛
#### 基本原則
- 輸入關鍵字 + `tab` 就可以轉換為 html tag
```js
div -> <div></div>
```
- `>` 可以產生巢狀子 tag
```js
div>h2

<div>
	<h2></h2>
</div>
```
- `+` 可以產生相臨 tag
```js
button+input
<button></button><input type="text">
```

還有一些其他的功能可看連結

[[技術分享] Emmet簡易教學 - 快速上手包](https://pjchender.blogspot.com/2016/07/emmet.html)

### 基本功能
html 的前處理語言，可以省略如 <> 等重複字元，而且比 emmet 更方便的是不用用 `tab` 來展開
